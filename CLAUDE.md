# LP リポジトリ — NEXTE マシンピラティス習得プログラム

高CVRなLPを作るためのナレッジベース + LP本体のリポジトリ。

## 構成

```
docs/BRIEF.md                     … 案件ブリーフ（商材・ターゲット・CV定義）※まずこれを読む
knowledge/
  01_lp-theory/                   … LP構成理論・CVR最適化の一般ノウハウ（PASONA/FV/CTA/EFO等）
  02_competitor/                  … フィットネス/ピラティス系LPの競合分析・コピー実例
  03_customer-insight/            … 見込み客の生の声・悩み・不安・決め手
  04_ad-and-measurement/          … 広告受け皿LP設計・LINE誘導CVフロー・GA4計測・法規制
lp/                               … LP本体（HTML/CSS/JS）
```

## ルール

- ナレッジファイルは `YYYY-MM-DD_タイトル.md` 形式、出典URL必須
- LP制作時は knowledge/ を参照し、根拠のある構成・コピーにする
- コピーで「必ず痩せる」「誰でも確実に」等の断定表現はNG（景表法）
- Before/After・体験談の扱いは knowledge/04 の法規制メモに従う
- CVはLINE経由の体験申込（30分2,200円）。CTAは全てLINE誘導に統一
