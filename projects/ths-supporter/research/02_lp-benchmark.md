# 高CVR LPの共通パターン10項目と本プロジェクトへの適用方針

前提: LP1=高齢者・施設向け（BtoC）、LP2=医療機関向け（BtoB）。衣料型・医療機器ではないため、表現上限は「サポート/支える/フィット/負担への配慮」レベル。

1. **FVは「症状名」でなく「生活シーンの不安」を言語化する**（バンテリン方式）
   → LP1: 「立ち上がる時・歩き出す時の股関節の不安に」等、痛みの断定を避けた動作シーンコピー。LP2は術後管理の課題提起型FV。
2. **メカニズムの可視化（構造名＋数値）で「効きそう」を作る**（プロ・フィッツ/ZW-7方式）
   → 3段階着圧（強・中・弱）の編地切替・カテーテル開口部を図解で1セクション。両LP共通の中核。
3. **試験データは「数値＋出典＋免責」のセット**（エンゼル方式）
   → n=63研究・n=62アンケートを研究デザイン明記＋「個人差があります」免責付きで提示。
4. **ターゲット別に「語彙の上限」を切り替える**（シグマックス方式）
   → LP1は「支える・寄り添う」、LP2は「術後管理の運用」文脈。ただし非医療機器なので両方とも「治療」「固定帯」は使わない。
5. **BtoCは購入CTA分散、BtoBは問い合わせ・資料CTA**
   → LP1: 注文ページCTAをFV直下・機能後・FAQ後の3箇所以上。LP2: 「注文ページ」＋「導入相談・サンプル問い合わせ」の2導線。
6. **サイズ不安の徹底解消がCVRの生命線**
   → 測り方ガイド＋「迷ったら1サイズ上」（カタログ記載事実）＋5サイズ展開の明示。LP2にはサイズ表と「1〜2サイズ上推奨」の運用知見（営業資料）を掲載。
7. **比較表で選択を単純化**
   → 1商品のため「一般的な巻くタイプとの構造の違い」を事実ベースの特徴表で提示（優良誤認にならない事実記述で）。
8. **権威・ヘリテージの借用**
   → 「富山の靴下メーカー助野の研究開発チーム『おたすけのLAB』開発」「販売は医療器械商社の村中医療器」という事実ベースの出自ストーリー。
9. **装着方法・運用面の図解**
   → LP1: 着脱・洗濯の説明。LP2: カテーテル留置時の装着手順4ステップ図＋病棟運用（洗い替え・サイズ説明のコツ）。
10. **薬機法・景表法セーフの定型句を全ページで統一**
    → OK「股関節まわりを支える」「歩行時の不安に寄り添う」／NG「治る」「痛みを予防」「転倒防止」「脱臼を防ぐ」「固定」。フッター免責を常設。
