# 03. CVRフレームワークの整理（LP設計に使う型）

最終更新: 2026-07-15。本プロジェクト（LP1=一般・施設向け / LP2=医療機関向け）への適用方針込み。

## 1. コピーライティング構成フレーム

### 新PASONA（神田昌典・2016改訂版）— LP1の骨格に採用
| 要素 | 内容 | LP1での実装 |
|---|---|---|
| **P**roblem（問題） | 悩みの明確化 | 悩み共感チェックリスト（立ち上がりのふらつき・長歩きの不安 等） |
| **A**ffinity（親近感） | 共感・寄り添い | 「動きたい気持ちはあるのに、不安が先に立つ」文脈。家族目線の一文も |
| **S**olution（解決策） | 解決の方向性 | 「履くだけの面サポート」という設計思想 |
| **O**ffer（提案） | 具体的オファー | 製品特徴・サイズ・購入方法（村中医療器オンライン） |
| **N**arrowing down（絞込） | 対象の限定 | 「筋力低下が気になり始めた方に」「施設の歩行支援備品に」 |
| **A**ction（行動） | 行動喚起 | 注文ページへのCTA＋サイズ選びガイド |

※旧PASONAの「Agitation（煽り）」は使わない — 高齢者向け・ヘルスケア領域では不安を煽る表現は薬機法/景表法リスクかつブランド毀損。

### AIDCA — LP2の骨格に採用（BtoB/医療者は Conviction が重い）
Attention（臨床課題の提起）→ Interest（製品コンセプト・構造）→ Desire（病棟運用上の利点）→ **Conviction（エビデンス・研究デザイン明記）** → Action（発注方法・問い合わせ）

## 2. ファーストビュー（FV）設計

- **3秒ルール**: FVで「誰の・何の・どうなる」が3秒で伝わること。スクロールせず第一CTAが見える
- FV構成要素: ①キャッチコピー（ターゲット明示） ②サブコピー（ベネフィット＋根拠の予告） ③ビジュアル（使用シーン） ④CTA ⑤信頼バッジ（メーカー・販売元・エビデンス有 等）
- LP1: スマホFVでコピー→画像→CTAが1画面に収まる設計。文字は大きく（高齢者・その家族）
- LP2: FVに「対象（THA等術後管理）」「エビデンスの存在」「発注導線」を明示

## 3. オファー・CTA設計

- **1LP=1オファー原則**: 出口は村中医療器オンライン注文ページ1つに集約（LP2のみ＋医療機関問い合わせの副導線）
- **CTAファースト**: FV内に第一CTA。以降、各説得ブロックの直後に配置（LP1は計4〜5回、LP2は3〜4回）
- **マイクロコピー**: ボタン周辺に不安低減の一言（「村中医療器オンラインストアへ」「サイズ表を見てから選べます」等）。クリック後の遷移先が外部ECであることを事前に伝え、期待と一致させる
- スマホは**下部固定の追従CTA**（親指到達圏・高さ44px以上）。ただし高齢者向けは誤タップ防止に閉じられる設計は不要、コンテンツを隠しすぎない高さに

## 4. 信頼・不安解消（ヘルスケアLPで最重要）

- エビデンスは「研究デザイン・対象・結果・限界」をセットで提示（誇張しない方が医療者・家族層のCVRは上がる）
- FAQで購入前の不安を潰す: サイズ選び／洗濯／医療機器かどうか／着け方／返品・交換
- 販売元の実在性: 村中医療器（医療器械商社）・製造元の助野（富山の靴下メーカー）を明記
- 注意書き: 「本品は医薬品・医療機器ではありません」をフッターに常設

## 5. モバイル最適化・技術

- モバイルファースト（LP1の閲覧は50代家族のスマホ検索が主と想定）
- 本文16px以上・行間1.8・コントラスト比4.5:1以上（LP1は18px基準）
- タップ領域44×44px以上、フォーム/リンクの間隔確保
- 画像はWebP＋width/height指定でCLS防止、遅延読み込み（FV画像はeager）
- ページ速度: 外部依存はGoogle Fontsのみ（display=swap）。JSは最小限。Lighthouse 90+目標
- 構造化データ: Product＋FAQPage（LP1）、MedicalWebPage系は使わず一般Article/FAQ（LP2・医療機器でないため）

## 6. 本プロジェクト固有の設計判断

1. 薬機法配慮がCVRを下げないよう、「効能の断定」の代わりに**「事実の具体性」**（3段階着圧の設計・カテーテル開口部・n=63研究の存在）で説得する
2. LP1は「本人」と「探している家族」の2読者を想定し、見出しは本人語り・補足は家族向け情報（サイズの測り方・贈り方）
3. LP2は感情表現を排し、表・数値・運用手順中心。「導入の流れ」を具体化して購買担当の稟議を通しやすくする
4. ビフォーアフター表現・体験談の捏造・医師推薦の演出は行わない（指示書5-2の禁止事項）
