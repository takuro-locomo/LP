# NEXTE マシンピラティス習得プログラム LP

案件フォルダ。リポジトリ全体のルールはルートの `CLAUDE.md` を参照。

## 構成

```
docs/BRIEF.md                     … 案件ブリーフ（商材・ターゲット・CV定義）※まずこれを読む
docs/image-prompts.md             … 画像生成プロンプト
docs/review-log.md                … レビュー記録
knowledge/
  02_competitor/                  … フィットネス/ピラティス系LPの競合分析・コピー実例
  03_customer-insight/            … 見込み客の生の声・悩み・不安・決め手
lp/                               … LP本体（HTML/CSS/JS）
reference/                        … 参考LP
scripts/upload_to_wp.sh           … WP固定ページ311へ下書き保存するスクリプト
```

※ LP理論・広告計測の共通ナレッジはルートの `knowledge/01_lp-theory/`・`knowledge/04_ad-and-measurement/` にある。

## ルール

- LP制作時は knowledge/（案件内＋ルート共通）を参照し、根拠のある構成・コピーにする
- コピーで「必ず痩せる」「誰でも確実に」等の断定表現はNG（景表法）
- Before/After・体験談の扱いはルート knowledge/04 の法規制メモに従う
- CVはLINE経由の体験申込（30分2,200円）。CTAは全てLINE誘導に統一
- 資格・認定は発行されない（FAQ・BRIEF参照）
