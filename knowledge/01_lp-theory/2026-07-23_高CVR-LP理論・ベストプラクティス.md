## 1. LP構成フレームワーク

| フレームワーク | 構造 | 向く商材・顧客温度感 |
|---|---|---|
| **新PASONA** | Problem → Affinity(親近感) → Solution → Offer → Narrow down(絞込) → Action | **悩み解決型商材**（ダイエット・美容・健康）。準顕在〜顕在層。旧PASONAの「煽り」を「寄り添い」に置換 |
| **QUEST** | Qualify(絞込宣言) → Understand(共感) → Educate(証拠) → Stimulate(ベネフィット) → Transition(行動) | **顕在層向け**。冒頭で「誰のためのページか」宣言。検索広告流入に強い |
| **AIDA/AIDMA** | Attention → Interest → Desire → (Memory) → Action | **低温度層**（ディスプレイ・SNS広告経由）。FVインパクト重視 |
| **BEAF** | Benefit → Evidence → Advantage → Feature | **EC・比較検討層**。結論先出し→証拠で固める |

- 使い分け: 広告経由(顕在)→QUEST/新PASONA、SNS/記事LP(潜在)→AIDA系、EC→BEAF
- ジム・美容など悩み起点商材は**新PASONAが鉄板。ただしFVはベネフィット先出し（BEAF的）のハイブリッドが主流**
- 新PASONAの「N（絞込）」=期間限定・人数限定オファーがCVRの直接ドライバー

## 2. ファーストビュー（FV）最適化

- **FVの平均離脱率60〜80%**。判断時間は**1〜3秒**（[ブルースクレイ](https://bruceclay.jpn.com/column/lp-qc7/)）
- 訪問者の70〜80%はFVだけで離脱or意思決定（[Infinity-Agent](https://infinity-agent.co.jp/lab/first-view/)）

**FVに置くべき5要素**:
1. キャッチコピー（2秒で理解できる）
2. メインビジュアル（サービスが視覚的に伝わる）
3. CTAボタン（above-the-fold CTAは下部配置比で最大+304%）
4. 権威づけ・実績（**数値化した実績を3つ**：例「満足度98%/症例3,000件/医師監修」）
5. オファー（初回価格・体験・特典）

補足: 左上ロゴ、TELはタップ発信+営業時間併記、スマホFVは高さ600〜700px以内。

**キャッチコピーの型**: ターゲット名指し型「◯◯でお悩みの40代女性へ」/ 数字型「2ヶ月で-8kg」/ ベネフィット先出し型（手に入る未来）/ 権威型。推奨フロー=**共感→ベネフィット→権威性**。

出典: [Cone](https://cone-c-slide.com/see-sla/blog/lp-first-view/) / [SiTest](https://sitest.jp/blog/?p=30645) / [トライハッチ](https://lab.tryhatch.co.jp/lp/first-view/) / [シンプリック](https://simplique.jp/lp-catchcopy-examples/)

## 3. CTA最適化

- 一人称コピー「Start **my** free trial」でクリック+90%（追試+10〜40%）（[Unbounce](https://unbounce.com/conversion-rate-optimization/how-to-write-a-call-to-action-that-converts-with-case-sudies/)）
- 「資料請求」→「無料で課題を解決する」でCVR 1.8倍（[KOHIMOTO](https://kohimoto.com/labo/web/system/18681/)）
- CTA周りコピー変更でCVR 1.9%→3.09%（[PLAN-B](https://www.plan-b.co.jp/blog/ad/71032/)）
- 色は**背景とのコントラスト**が本質（コントラスト強化で最大+34%）。赤/オレンジ系+5〜21%の報告（[WiserReview](https://wiserreview.com/blog/call-to-action-statistics/) / [Flint](https://www.flint.com/blog/landing-page-cta-button-performance-statistics)）
- CTA(ゴール)は1LPにつき**1種類**。A/BテストでCTA最適化した場合平均+28%（[Martal](https://martal.ca/cta-best-practices-lb/)）

**実務公式**:
- 文言 = **動詞 + ベネフィット + ハードル低減**。NG「送信」→OK「まずはLINEで相談（30秒）」
- **マイクロコピー**が費用対効果最強:「※しつこい勧誘は一切ありません」「残り◯枠」「30秒で完了」
- **スマホは追従CTA必須**（画面下部・親指ゾーン）
- 「無料」を含む文言はクリック率UP

出典: [アナグラム](https://anagrams.jp/blog/how-to-write-microcopy/) / [microcopy.org](https://microcopy.org/basic_knowledge/cta_reason/) / [SiTest](https://sitest.jp/blog/?p=30661) / [HubSpot](https://blog.hubspot.jp/marketing/14-real-life-examples-of-cta-copy-you-should-copy)

## 4. EFO（フォーム最適化）／ LINE誘導

- 平均**約68%がフォーム途中離脱**。EFOでCVR20〜50%改善事例多数（[uマーケティング](https://umarketing.co.jp/column/efo/)）
- HubSpot: 4→3項目でCVR約+50%。1項目削除で+26%（Neil Patel）。目安**1項目削減=CV+3〜5%**（[Venture Harbour](https://ventureharbour.com/how-form-length-impacts-conversion-rates/)、反証は[CXL](https://cxl.com/blog/reduce-form-fields/)）
- 単一ページフォーム完了率4.53% vs **マルチステップ13.85%**

**LINE誘導 vs フォーム**:
- **LINE友だち追加はフォーム一体型の2〜3倍以上のCVR**事例あり（入力不要・ワンタップ）（[株式会社禅](https://zen-web.co.jp/column/column-lp-cvpoint/)）
- LINE=低温度層・スマホ主体・ナーチャリング前提 → **ジム/ピラティスと好相性**
- 注意: **友だち追加→予約の2段目歩留まり設計が必須**（挨拶メッセージ・リッチメニュー・自動応答）

出典: [formrun](https://form.run/home/blog/efo/design/improvement) / [LANY](https://www.lany.co.jp/blog/efo-cvr) / [増客くん](https://zokyaku.com/column/6862)

## 6. モバイル最適化・表示速度

- 読み込み1秒 vs 5秒でCVR **2.5倍**差。遅延1秒ごとにCV約-7%
- モバイルは**3秒で53%離脱**（Google）
- 目標: **LCP<2.5s / INP<200ms / CLS<0.1**
- 即効策: FV画像WebP化+`fetchpriority="high"`、ヒーロー以外遅延読込、Webフォント削減
- ジム系LPは流入8〜9割スマホ → **スマホファースト設計**

出典: [Kanuka](https://www.kanukadigital.com/2025/09/website-load-time-statistics-in-2025/) / [Edmonds](https://edmondscommerce.co.uk/research/performance/load-time-impact/) / [NitroPack](https://nitropack.io/blog/how-page-speed-affects-conversion/)

## 7. 信頼要素

- 95%が購入前にレビューを読む。**CTA近くの推薦文/バッジで+15〜34%**
- **動画テスティモニアルでCV+80%**
- 返金保証: 30日保証明示で売上+21%。CTA直下の保証表示でCV+30%
- **配置の鉄則: 信頼要素は「CTAの直前・直上」**（決断の瞬間に効かせる）
- 優先順位: ①写真付き実名口コミ → ②数値実績（FVに3つ）→ ③第三者認証 → ④保証

## まとめ — NEXTE LPへの適用優先順位

1. **FV**: ターゲット名指し+ベネフィット先出しコピー / 実績3つ / オファー / FV内CTA
2. **CTA**: 「LINEで体験を予約する（30秒）」型 + マイクロコピー + スマホ追従ボタン
3. **CV手段**: LINE誘導一本化（フォーム比2〜3倍）
4. **信頼**: 口コミ・実績をCTA直前に配置
5. **速度**: LCP2.5秒以内・スマホファースト
6. **構成**: 新PASONA × FVベネフィット先出しハイブリッド
