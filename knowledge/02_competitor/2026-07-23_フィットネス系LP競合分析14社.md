# 総括：高CVRフィットネス系LPの共通パターン

## FVの勝ちパターン
- **「情緒コピー＋機能サブコピー＋実績数字」の3点セット**
- コピー3系統: 結果保証型 / 心理ハードル低減型 / 世界観・自己変革型
- ターゲット限定コピーが強い（「30歳からの」「女性専用」）— 絞るほど刺さる

## オファー設計の定石
- ピラティス系標準は「体験0円」、ジム系は「無料カウンセリング」（14中11が無料）
- **無料・割引時は通常価格を併記して打ち消す**
- 期限・条件: 月替わり/日付明記/乗り換え割
- リスクリバーサル: 返金保証/一生サポート

## 鉄板の構成順序
```
FV（コピー＋実績数字＋CTA）
→ 悩み共感/課題提示
→ 選ばれる理由（3〜6個）
→ Before/After・お客様の声（年代別）
→ 料金（＋比較表）
→ 体験までの流れ（STEP1〜4）
→ FAQ
→ CTA再掲＋アクセス
```
- CTAは最低3箇所+LINE追従ボタンが近年標準
- 地方LPはGoogle口コミをFV直下に

## CTA文言
- 最頻出「無料カウンセリング予約」「無料体験予約はこちら」
- 「LINEで予約」併設ほぼ必須（電話/WEB/LINEの3導線）
- 行動を軽くする修飾:「まずは」「30分の」

## 料金の見せ方
| 型 | 例 | 使いどころ |
|---|---|---|
| 総額正直型 | RIZAP | 保証で正当化できる高単価 |
| 1回単価分解型 | 24/7等 | 中価格帯の割安感 |
| 分割月額型 | BEYOND長野「月々4,800円〜」 | 地方・高額コース |
| アンカー型 | the SILK 1,980円 | サブスク通い放題 |
| 非開示送客型 | CLUB PILATES | 体験CVに全振り |

## 信頼要素スタック順（強い順）
1. 実績数の具体化（端数まで出すと信憑性UP）
2. 第三者評価（アワード・メディア・Google口コミ）
3. Before/After（数値+年代+免責セット）
4. 専門資格・監修
5. 人の顔（実名・顔写真・実績）
6. 心理的安全性の数字（「初心者比率80%」等）

## NEXTE（地方LP）への示唆
- 地方は**Google口コミ・地元メディア・実名トレーナー**が大手の実績数の代替
- 「体験通常価格→期間限定割引」打ち消し表示+「入会金0円」同時提示が再現性最強
- CTAは「無料カウンセリング」より「**LINEで簡単予約**」の方が地方・女性向けで摩擦が低い
- **「習得型・卒業型」ポジションは14社中どこも取っていない空白地帯**（最接近はpilates Kの名言「30回ですべてが変わる」とCLUB PILATESの養成講座のみ）→ NEXTEの差別化軸として有効
